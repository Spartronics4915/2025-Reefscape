# Control Map

## Driver controller

| input   | action |
| -------- | ------- |
| Left stick |  Drive |
| Push Left Stick |  reset FR pos heading |
| Right stick | Field angle |
| L & R Triggers | X Brake Swerve |
| R Trigger | reef auto orient |
| B | Toggle field relative |
| A | Set FR pos heading |
